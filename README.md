# Wordle Solver
Solve worldle with python. Script runs through 2,499 five letter words (https://www.kaggle.com/datasets/cprosser3/wordle-5-letter-words) and tries to solve within six tries. 

Average Tries:  3.86

Average Words:  3.92

Percent Solved: 0.98

#
# Run
`poetry run python app/solver.py`

# Run Tests
`poetry run pytest`
